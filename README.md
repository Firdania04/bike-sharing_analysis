# Dicoding Collection Dashboard

## How to Run This Project ?

## install all library

```
pip install numpy pandas matplotlib seaborn jupyter streamlit babel
```

or

```
pip install -r requirements.txt
```

Go to dashboard folder

```
cd dashboard

```

## Run steamlit app

```
streamlit run dashboard.py
```
